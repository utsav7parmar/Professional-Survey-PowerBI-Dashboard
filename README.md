# 📊 Professional Survey Power BI Dashboard

## 🚀 Overview

This project presents an **interactive Power BI dashboard** developed using survey data collected from **data professionals worldwide**. The dashboard provides valuable insights into salary trends, job roles, programming language preferences, work-life balance satisfaction, salary satisfaction, and challenges faced while entering the data industry.

The objective of this project is to transform raw survey data into meaningful business insights through effective data visualization and dashboard design.

## 🎯 Key Insights

### 👥 Survey Participants

* **Total Survey Takers:** 630
* **Average Age:** 29.87 Years

### 💰 Salary Analysis

* Data Scientists reported the highest average salary.
* Data Engineers and Data Architects followed closely.
* Students and entry-level professionals showed comparatively lower salary ranges.

### 🌍 Geographic Distribution

Survey responses were collected from:

* United States
* India
* Canada
* United Kingdom
* Other Countries

### 💻 Programming Language Preferences

Most preferred programming languages among survey participants:

1. Python
2. R
3. Other Languages
4. C/C++
5. JavaScript
6. Java

### 🚧 Difficulty Entering the Data Industry

Respondents reported different levels of difficulty when entering the data field:

* Easy
* Very Easy
* Neutral
* Difficult
* Very Difficult

### 😊 Satisfaction Metrics

#### Work-Life Balance

**Average Rating:** 5.74 / 10

#### Salary Satisfaction

**Average Rating:** 4.27 / 10
